# OpenGL-3DCharacter

## Features

🦴 Skeletal Animation System using bone matrices and keyframe interpolation

🧍 Third-person character controller (WASD to move, mouse to look, Q/E to adjust camera height)

🎬 Multiple animation states

  Standing
  
  Walking
  
  Dancing (press K)
  
  Dying (press L)

🏞️ 3D ground plane for spatial context

## Controls Key	Action
  
  W / A / S / D	Move character
  
  K	Play “Silly Dancing” animation
  
  L	Play “Dying” animation
  
  Q / E	Raise or lower camera height
  
  Mouse	Look around
  
  Esc	Exit program
